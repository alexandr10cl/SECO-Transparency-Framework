Transparency Evaluation Framework for Software Ecosystems Portals

A technology solution to analyze, evaluate, and enhance transparency in software ecosystem portals. Create an account now to start assessing your portal!

About the Project

SECO-TransP is a framework designed to evaluate the transparency of software ecosystem (SECO) portals. It guides portal managers and developers through the configuration and execution of assessments based on common ecosystem practices, such as access to documentation, source code, governance information, and communication channels. From the selection of relevant procedures, the framework generates specific tasks that help assess the openness and clarity of the information provided by a SECO portal.


During the evaluation process, participating developers receive a unique access code to interact with the evaluated SECO portals and perform practical tasks. After each task, they provide quick feedback, and upon completing each procedure, they answer detailed Key Success Criteria (KSC) questions based on transparency guidelines. The responses are analyzed quantitatively using a method inspired by the System Usability Scale (SUS), ensuring a clear interpretation of the results.


At the end of the evaluation, SECO-TransP consolidates the data into an interactive dashboard that displays performance by procedure, guideline, and evaluation dimension. Additionally, the framework highlights critical issues and suggests improvements, making it a valuable tool for the continuous enhancement of transparency in SECO portals. With SECO-TransP, managers and developer communities gain a solid foundation to assess and improve developer experience and ecosystem governance.

Framework Tools

Discover the integrated solutions that comprise the SECO-TransP Framework.

Browser Extension

A browser extension that guides developers through tasks, collects interaction data, and captures developers experience (DX) feedback to assess portal transparency.

Dashboard

Delivers powerful insights into analyzed portals with a comprehensive transparency report based on established guidelines.

UX-Tracking

A tool that collects real-time user interaction data, maps user journeys, identifies key touchpoints, and delivers valuable user experience (UX) insights for both web and mobile environments.

How the SECO-TransP Evaluation Works

Learn how to use our tools to assess the transparency of your software ecosystem portal.

1. Create Evaluation

After creating an account, you can start a new assessment for your portal by selecting the specific common ecosystem procedures to be tested by developers. All you need to do is provide the link to your portal.

2. Data Collection

When an assessment is created, a code will be generated so developers can join it through the browser extension. The extension will record their interactions with the portal, collecting relevant information for the evaluation.

3. Data Analysis

Access the dashboard to explore the collected data, generate detailed reports, and view performance charts. Use these insights to understand usage patterns and identify opportunities to improve your portal’s transparency.

4. Correction

Review the generated report, go over the recommended transparency guidelines, and implement the necessary corrective actions to enhance your portal based on the identified issues.
